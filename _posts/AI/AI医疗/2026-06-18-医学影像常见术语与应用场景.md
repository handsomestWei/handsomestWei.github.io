## 一、背景：首页在讲什么

[RadiAnt](https://www.radiantviewer.com/zh/) 定位为 **医学图像的 PACS / DICOM 浏览器**（Windows 桌面阅片软件），强调：

- 打开院内常见的 **DICOM** 研究（Study）
- 提供 **窗宽窗位、测量、MPR、3D、融合** 等阅片工具
- 可作为 **PACS 客户端** 检索、下载研究，并支持 **本地存档、导出、脱敏** 等

以下按首页板块顺序整理 **术语 + 通俗解释 + 典型场景**。版本新闻、论坛帖等 **未全文收录**。

## 二、核心系统名词

### DICOM

| 官网语境 | 普及解释 |
|----------|----------|
| 医学数字成像与通信标准；`.dcm` 文件 | 院内 CT/MRI/X 线等 **标准影像文件格式**，除像素外还带 **患者、检查、序列、层厚、窗宽窗位** 等大量标签。 |

### PACS（Picture Archiving and Communication System，医学影像存档与通信系统）

| 官网语境 | 普及解释 |
|----------|----------|
| 从 PACS 位置搜索和下载研究；PACS 客户端 | 医院 **影像数据中心 + 调阅网络**。阅片工作站、RadiAnt 等可作为 **客户端**，用 DICOM 网络协议 **查询（C-FIND）、拉取（C-MOVE）** 某次检查的全部序列。 |

**首页提到的 DICOM 网络角色（了解即可）**

| 缩写 | 含义（简化） |
|------|----------------|
| C-ECHO SCU/SCP | 连通性测试（「对方在不在线」） |
| C-FIND SCU | 按条件 **查询** 研究列表 |
| C-MOVE SCU | **把研究搬到本机** |
| C-STORE SCP | **接收** 别人推过来的 DICOM |

**场景**：医生不在固定工作站前，用笔记本 + RadiAnt 连医院 PACS 拉 **胸部 CT** 到本地查看。

### 研究（Study）/ 序列（Series）/ 实例（Instance）

| 术语 | 普及解释 |
|------|----------|
| **研究 Study** | 一次检查预约产生的一套数据，如「2026-06-03 胸部 CT」 |
| **序列 Series** | 同一检查下不同扫描方式，如 **平扫 / 增强 / 肺窗重建**、MRI 的 **T1 / T2 / FLAIR** |
| **实例 Instance** | 通常 **一张切片 = 一个 DICOM 文件**（CT/MRI 常见） |

**场景**：首页「不同系列或研究的比较」——同时打开 **平扫与增强** 两个 Series，或对比 **两次不同日期** 的 Study。

## 三、多模态 DICOM 类型

官网列出软件可打开的 **Modality（模态）** 代码。下表：**中文名 | 代码 | 是什么 | 常见场景**。

| 中文（官网） | DICOM 代码 | 是什么 | 典型场景 |
|--------------|------------|--------|----------|
| 数字式放射照相术 | **CR, DX** | 数字化 X 线片（板/detector） | 胸片、骨骼片 |
| 乳房 X 射线照相术 | **MG** | 乳腺钼靶 | 乳腺癌筛查 |
| 计算机断层照相法 | **CT** | X 线断层，一层层切片 | 胸腹急诊、肺结节、骨折 |
| 磁共振 | **MR** | 磁共振断层/多序列 | 颅脑、关节软组织 |
| 正电子发射计算机断层扫描 PET-CT | **PT** | PET 代谢 + CT 解剖 | 肿瘤分期、代谢灶 |
| 超声波扫描术 | **US** | 超声图像/视频 | 腹部、心脏、产科 |
| 数字血管造影术 | **XA** | X 线透视/造影序列 | 心血管介入、DSA |
| 核医学照相术 | **NM** | 核医学平面或断层 | 骨扫描、部分 SPECT |
| 二次图片和扫描的图像 | **SC** | 从胶片等 **二次采集** 的图 | 老片数字化 |
| 结构化报告 | **SR** | **结构化** 文字/测量报告 | 挂接在影像上的报告数据 |

### 图像形态（官网列举的类型）

| 类型 | 含义 | 例子 |
|------|------|------|
| **单色** | 灰度图 | CR、CT、MR |
| **彩色** | 彩色显示 | 部分 US、3D 伪彩 |
| **静态** | 单帧或单张 | CR、MG、CT 单层 |
| **连续动态** | 多帧时间序列 | XA 透视、US 动态 |
| **未压缩 / 压缩** | 存储方式 | RLE、JPEG、JPEG 2000 等 |

**与 AI 的关系**：训练前需知 **模态代码** 与 **是否多帧**；不能把 US 视频当 CT 单层随便混用。

## 四、阅片基础操作

首页列出的操作对应临床/科研 **日常阅片动作**，术语如下。

| 功能（官网） | 普及解释 |
|--------------|----------|
| 镜头远近调节和摇摄 | 缩放、平移浏览大图或厚层 |
| 亮度与对比度、反色 | 相当于调 **窗宽窗位** 或显示曲线 |
| 预置 CT 窗口（肺、骨等） | **窗宽窗位预设**：肺窗看肺纹理，骨窗看骨折 |
| 旋转 / 翻转 | 调整显示方向 |
| 分段长度 | 测量距离（如结节径线） |
| 圆形/椭圆形及密度统计 | 画 ROI，CT 上可读 **HU 最大/最小/平均** |
| 角度（含科布角） | 脊柱侧弯等角度测量 |
| 自由画笔 | 标注、勾画示意 |

**场景**：读 CT 肺结节—— **肺窗** 观察 → **椭圆 ROI** 测大小与平均 HU → 截图写入报告（常再 **导出 JPEG**）。

## 五、工作流场景

### 5.1 不同系列或研究的比较

- **同 Study 多 Series**：如 CT **造影前 / 造影后**，可 **自动同步** 滚动位置。
- **不同解剖方位**：如 MRI 轴位 / 冠位，用 **交叉参考线（Crosshair）** 对齐解剖位置。

**场景**：对比增强前后 **同一层** 是否出现强化。

### 5.2 从 PACS 搜索和下载

- 在 PACS 上 **查询患者 → 选中 Study → 下载到本地** 临时或本地库。

**场景**：会诊医生从分院 PACS 拉取 **外院 CT**。

### 5.3 本地存档

- 从 **CD/DVD、U 盘、文件夹、PACS** 导入并 **长期保存在本机**，避免光盘损坏或反复 C-MOVE。

**场景**：病人带来 **检查光盘** 后，医院导入 PACS 前先在本地留档。

### 5.4 DICOM 转 JPEG / BMP / WMV

- 导出用于 **PPT、论文插图、教学视频**；**会丢失** 大量 DICOM 元数据与 HU 精确性。

**场景**：科室会 **导出 JPG** 讨论，但科研与 AI 应优先保留 **DICOM 或 NIfTI**。

### 5.5 快捷键 CTRL+C

- 当前显示图像复制到剪贴板，贴入 Word/PPT。

### 5.6 病人 CD/DVD 自动播放包

- 免安装 Autorun 式 **DICOM 浏览器** 放在光盘上，适合 **外院带片** 场景（官网产品卖点）。

## 六、高级显示与后处理术语

### 6.1 MPR（多维重建 / 多平面重建）

| 项 | 说明 |
|----|------|
| **是什么** | 由 **轴位 CT/MRI 体数据** 在软件中实时算出 **冠状面、矢状面、斜面** |
| **官网例子** | 2000+ 张轴位 CT 约数秒可重建冠状位（硬件依赖） |
| **场景** | 不看单张切片，而看 **矢状面** 观察脊柱序列 |

### 6.2 3D VR（三维体绘制）

| 项 | 说明 |
|----|------|
| **是什么** | 把 CT/MR **体数据** 当作半透明 **三维体** 旋转观察 |
| **工具** | 调颜色/不透明度、 **手术刀切割** 隐藏遮挡 |
| **场景** | 骨科 **骨折块空间关系**、血管畸形概览（非诊断级表述需由医生判断） |

### 6.3 PET-CT 图像融合

| 术语 | 普及解释 |
|------|----------|
| **PET-CT** | **PET（代谢）** 叠在 **CT（解剖）** 上 |
| **FDG** | 常用示踪剂；代谢增高区在彩色 PET 上更亮 |
| **SUVbw** | 按体重标准化的摄取值；椭圆 ROI 可算 **最大/最小/平均 SUV** |
| **扩展** | 官网称也可用于 **MR 与 DWI 与 T1/T2 融合** 等 |

**场景**：肿瘤 **葡萄糖代谢增高灶** 在 CT 上对应哪一段肺/淋巴结。

### 6.4 DSA（数字减影血管造影）

| 项 | 说明 |
|----|------|
| **模态** | 多用于 **XA（透视）** 序列 |
| **原理** | **造影后图 − 造影前掩膜** → 减掉骨与软组织， **血管更清晰** |
| **像素偏移** | 自动对齐掩膜与 live 图，减少运动伪影 |

**场景**：介入科查看 **冠状动脉或外周动脉** 充盈情况。

### 6.5 TIC（时间-强度曲线）

| 项 | 说明 |
|----|------|
| **是什么** | 动态增强扫描中，某 ROI **信号随时间变化** 的曲线 |
| **典型场景** | **乳腺 MRI** 动态增强，鉴别良恶性倾向（由放射科医师综合判断） |
| **曲线分型（官网）** | **Ia** 持续上升 / **Ib** 后期趋平 / **II** 平台 / **III** 中期流出（流出型） |
